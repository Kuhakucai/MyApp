# 📸 美甲款式图片使用说明

## 当前图片来源

所有美甲款式图片目前使用 **Unsplash** 免费图片服务，这是一个提供高质量免费图片的平台。

## 图片URL格式

```
https://images.unsplash.com/photo-[图片ID]?w=400&h=300&fit=crop
```

参数说明：
- `w=400`: 图片宽度400px
- `h=300`: 图片高度300px  
- `fit=crop`: 裁剪模式

## 🔄 如何替换为自己的图片

### 方案一：使用本地图片（推荐）

1. **准备图片**
   - 建议尺寸：400×300px 或 800×600px
   - 格式：JPG 或 PNG
   - 大小：每张控制在 200KB 以内

2. **放置图片**
   ```
   images/
   ├── products/
   │   ├── nail-1.jpg
   │   ├── nail-2.jpg
   │   └── ...
   └── banners/
       ├── banner-1.jpg
       └── ...
   ```

3. **修改代码**
   
   编辑 `data/products.js`:
   ```javascript
   {
     id: 1,
     name: '星空渐变美甲',
     image: '/images/products/nail-1.jpg',  // 本地路径
     // ...
   }
   ```

### 方案二：使用图床服务

推荐的免费图床：

1. **七牛云**
   - 免费额度：10GB存储 + 10GB流量/月
   - 网址：https://www.qiniu.com/

2. **腾讯云COS**
   - 免费额度：50GB存储 + 10GB流量/月
   - 网址：https://cloud.tencent.com/product/cos

3. **阿里云OSS**
   - 适合商用，稳定性高
   - 网址：https://www.aliyun.com/product/oss

使用步骤：
```javascript
// 1. 上传图片到图床
// 2. 获取图片URL
// 3. 替换代码中的图片链接

image: 'https://your-cdn.com/nail-images/star-gradient.jpg'
```

### 方案三：继续使用Unsplash

**优点**：
- 完全免费
- 图片质量高
- 自动CDN加速

**缺点**：
- 需要网络访问
- 图片不是自己的实际产品

**如何查找更多美甲图片**：

1. 访问 https://unsplash.com/
2. 搜索关键词：
   - nail art
   - manicure
   - nail polish
   - nail design
   - french nails

3. 点击喜欢的图片，复制URL
4. 修改URL格式：
   ```
   原始: https://unsplash.com/photos/xxxxx
   使用: https://images.unsplash.com/photo-xxxxx?w=400&h=300&fit=crop
   ```

## ⚠️ 注意事项

1. **版权问题**
   - Unsplash图片可商用，但建议使用自己的实拍图
   - 如使用其他图片，请注意版权

2. **图片大小**
   - 每张图片建议不超过200KB
   - 过大会影响小程序加载速度

3. **微信小程序限制**
   - 小程序包大小限制：主包2MB，分包各2MB
   - 建议使用CDN，不要将大量图片打包在小程序内

4. **网络图片域名配置**
   - 在微信小程序后台配置服务器域名
   - 开发管理 → 开发设置 → 服务器域名 → downloadFile合法域名
   - 添加：`https://images.unsplash.com`（如使用Unsplash）
