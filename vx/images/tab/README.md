# TabBar 图标说明

## 需要的图标文件

请在此目录下放置以下8个图标文件（尺寸建议：81x81px）：

### 首页
- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）

### 点餐
- `menu.png` - 点餐图标（未选中）
- `menu-active.png` - 点餐图标（选中）

### 订单
- `order.png` - 订单图标（未选中）
- `order-active.png` - 订单图标（选中）

### 我的
- `mine.png` - 我的图标（未选中）
- `mine-active.png` - 我的图标（选中）

## 图标要求

1. **尺寸**：81x81 像素
2. **格式**：PNG 格式（支持透明背景）
3. **颜色**：
   - 未选中状态：灰色 (#999999)
   - 选中状态：主题色 (#FF6B35)

## 获取图标的方式

### 方式1：使用 Iconfont（推荐）
1. 访问：https://www.iconfont.cn/
2. 搜索关键词：home、menu、order、user
3. 下载合适的图标
4. 使用图片编辑软件调整尺寸和颜色

### 方式2：使用微信开发者工具
1. 在微信开发者工具中可以临时使用默认图标
2. 打开小程序后，即使没有图标文件也能运行（会显示默认图标）

### 方式3：在线图标工具
- IconKitchen: https://icon.kitchen/
- Canva: https://www.canva.com/

## 临时解决方案

如果暂时没有图标，小程序依然可以运行，只是底部导航栏会显示空白或默认图标。
