## 🚀 快速开始

### 1. 安装微信开发者工具
下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本项目目录：`e:\软件工具\vx`
4. AppID 选择「测试号」或填入你的小程序 AppID
5. 点击「导入」

### 3. 准备图标（可选）
- TabBar 需要图标文件，请参考 `images/tab/README.md`
- 如果暂时没有图标，小程序也能正常运行

### 4. 运行预览
- 点击「编译」按钮
- 在模拟器中查看效果
- 扫描二维码在手机上真机测试

## 🎨 技术栈

- **开发方式**：原生微信小程序
- **数据存储**：LocalStorage（本地缓存）
- **UI设计**：粉色系渐变，时尚优雅界面
- **主题色**：#FF69B4（浪漫粉）

## 💡 核心功能说明

### 数据存储
项目使用微信小程序的 LocalStorage 进行数据存储：
- **购物车数据**：存储在 `cart` 键
- **预约数据**：存储在 `orders` 键
- **美甲款式数据**：硬编码在 `data/products.js`

### 购物车逻辑
- 添加服务自动合并相同项目
- 实时计算总价
- 支持数量调整和删除
- 满减优惠自动计算

### 预约系统
- 自动生成预约单号（时间戳+随机数）
- 预约状态：待处理、制作中、已完成、已取消
- 预约详情完整展示

## 🔧 自定义配置

### 修改门店信息
编辑 `app.js` 文件中的 `globalData.shopInfo`：
```javascript
shopInfo: {
  name: '你的美甲店名称',
  address: '店铺地址',
  phone: '联系电话',
  openTime: '营业时间'
}
```

### 修改美甲款式数据
编辑 `data/products.js` 文件：
- 修改 `categories` 数组添加分类
- 修改 `products` 数组添加/编辑美甲款式

### 修改主题色
编辑 `app.wxss` 文件中的颜色变量：
```css
.primary-color {
  color: #你的颜色;
}
```

## 📝 开发说明

### 添加新款式
在 `data/products.js` 的 `products` 数组中添加：
```javascript
{
  id: 20,                    // 唯一ID
  name: '款式名称',
  desc: '款式描述',
  price: 128.0,             // 价格
  image: '图片URL',
  category: 2,              // 分类ID
  hot: true,                // 是否热门（可选）
  sales: 100                // 销量（可选）
}
```

### 修改满减规则
在 `pages/cart/cart.js` 的 `loadCart` 方法中修改：
```javascript
if (subtotal >= 100) {
  discount = 15  // 满100减15
} else if (subtotal >= 50) {
  discount = 5   // 满50减5
}
```

## ⚠️ 注意事项

1. **AppID配置**：测试时可使用测试号，正式发布需要注册小程序
2. **图标准备**：TabBar图标需要自行准备或使用占位图
3. **数据持久化**：当前使用LocalStorage，清除缓存会丢失数据
4. **支付功能**：在线支付需要企业主体小程序才能使用
5. **真机测试**：部分功能需要在真机上测试效果
