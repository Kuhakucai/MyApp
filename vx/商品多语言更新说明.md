### **3. 首页商品显示** ✅

更新了首页热门商品的加载逻辑：

```javascript
loadProducts() {
  const currentLang = i18n.getCurrentLanguage()
  const hotProducts = products.filter(item => item.hot).slice(0, 4).map(product => {
    const translation = productsTranslations[product.id]
    if (translation) {
      return {
        ...product,
        name: translation.name[currentLang] || product.name,
        desc: translation.desc[currentLang] || product.desc
      }
    }
    return product
  })
  
  this.setData({ hotProducts })
}
```

**效果**：首页热门款式的名称和描述会根据当前语言动态显示

```javascript
loadProducts() {
  const currentLang = i18n.getCurrentLanguage()
  const translatedProducts = products.map(product => {
    const translation = productsTranslations[product.id]
    if (translation) {
      return {
        ...product,
        name: translation.name[currentLang] || product.name,
        desc: translation.desc[currentLang] || product.desc
      }
    }
    return product
  })
  
  this.setData({ products: translatedProducts })
}
```

**效果**：选款页所有19个商品的名称和描述都会翻译

## ✨ 技术特点

### **1. 动态加载**
- 根据当前语言动态加载翻译
- 语言切换后自动更新

### **2. 回退机制**
```javascript
name: translation.name[currentLang] || product.name
```
如果翻译不存在，自动回退到原始中文

### **3. 可扩展性**
添加新语言只需：
1. 在 `products-i18n.js` 中添加新语言代码的翻译
2. 在 `i18n.js` 中添加新语言配置
3. 无需修改其他代码
